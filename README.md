# SSTS Bug Tracker
Django web app for management of bug tickets 

### Application Features
- Create a bug ticket
- View all bug tickets
- View individual bug ticket
- Update bug ticket
- Remove bug ticket
- Create comment
- View comment
- Authenticate the user (Login)
- Logout
